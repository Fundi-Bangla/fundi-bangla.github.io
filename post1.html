GitHub দিয়ে মোবাইল ব্লগ তৈরি

আপনি কি মোবাইল দিয়ে নিজের ব্লগ বানাতে চান? এই পোস্টে আমি দেখাবো কিভাবে আপনি GitHub ব্যবহার করে একদম ফ্রি তে একটি ব্লগ তৈরি করতে পারবেন।

ধাপ ১: GitHub অ্যাকাউন্ট খুলুন

প্রথমে github.com এ যান এবং একটি ফ্রি অ্যাকাউন্ট খুলুন।

ধাপ ২: Repository তৈরি করুন

নতুন একটি Repository খুলুন, নাম দিন: yourusername.github.io

ধাপ ৩: HTML ফাইল আপলোড করুন

আপনার তৈরি করা index.html ফাইলটি Upload করুন।

ধাপ ৪: GitHub Pages চালু করুন

Settings > Pages > Deploy from Branch সিলেক্ট করুন এবং সাইট চালু করুন।

ব্যাস! আপনার ওয়েবসাইট এখন লাইভ!

আরও বিস্তারিত জানতে বা কাস্টম ডিজাইন পেতে আমাদের হোম পেজে ফিরে যান।
